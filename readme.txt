Plugin zum Einbinden externer Inhalte in WordPress.

== Description ==
Plugin zum Einbinden externer Inhalte in WordPress. Nach der Aktivierung können externe Inhalte in Seiten bzw. Beiträgen integriert werden. Dazu wird im Inhaltsbereich mit dem WordPress-Editor folgender Shortcode eingefügt: [wpiec]http://www.meine-seite.de/meine-datei.html[/wpiec]  (Die URL ist durch die richtige Web-Adresse zu ersetzen)

== Installation ==
1. Upload wp-insert-external-content`  to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
